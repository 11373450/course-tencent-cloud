## 酷瓜云课堂

#### 项目介绍

酷瓜云课堂，依托腾讯云基础服务架构，采用C扩展PHP框架Phalcon开发，致力开源网课系统，开源网校系统，开源在线教育系统。

#### 系统功能

实现了点播、直播、专栏、会员、微聊等，是一个完整的产品，具体功能我也不想写一大堆，自己体验吧！

友情提示：

- 系统配置低（1核 1G 1M 跑多个容器），切莫压测
- 课程数据来源于网络（无实质内容），切莫购买
- 管理后台已禁止数据提交，私密配置已过滤
